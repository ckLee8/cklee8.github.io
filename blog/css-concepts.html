Positioning

What are the differences between relative, absolute, fixed, and static positioning?

Relative positioning is type of formatting applied to a webpage element (ie: picture, paragraph) that allows for its place on the webpage to be influenced by other relative elements. Absolute positioning, on the other hand, is a type of formatting that makes a webpage element completely unaffected by other web elements. The formatting parameters that are set are in a sense absolute. Fixed is another useful positioning format that allows for items to stay in a specific place on the screen reguardless of where the rest of the webpage is. For instance, if you place an image in the top right corner in fixed positioning, and you scroll down the webpage, the item should stay in the corner of your screen. Lastly, static positioning is the default formatting for page elements. The elements will flow in the order presented in the code.
